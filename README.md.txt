# SQL E-Commerce Sales Analysis

This project demonstrates a simple e-commerce database built with MySQL and queried using basic SQL commands.

## 📁 Files

- `1_schema.sql` – Contains table definitions and sample data inserts.
- `2_queries.sql` – Contains 4 analytical SQL queries.
- `3_results.txt` – (Optional) Contains the output results from each query, copied from dbfiddle.

## 📊 Queries Performed

1. Customer orders with product details.
2. Total spending by each customer.
3. Product-wise quantity sold.
4. Monthly sales summary.

## 🔧 Tested On

- [dbfiddle.uk](https://dbfiddle.uk)
- MySQL 8.0
